**Bus Ticket System**

This is a full-stack Bus Ticket Management System built using:<br />
Spring Boot (Backend + JWT Authentication)<br />
React Native (Passenger App + Collector App)<br />
React JS (Admin Dashboard)<br />
PostgreSQL (Database)<br />
<br />
<br />
**How to Run**<br />
<br />
Backend (Spring Boot)<br />
cd SpringBoot/Backend<br />
mvn spring-boot:run<br />
<br /><br />
Passenger App (React Native)<br />
cd passenger-app<br />
npm install<br />
npm start or <br />
npx expo start (to launch it on phone)<br />
(Install Expo go on your phone and scan qr code from the terminal)<br />
<br /><br />
Collector App (React Native)<br />
cd collector-app<br />
npm install<br />
npm start or npx expo start (to launch it on phone)<br />
(Install Expo go on your phone and scan qr code from the terminal)<br />
<br /><br />
Admin Dashboard (React)<br />
cd SpringBoot/FrontEnd<br />
npm install<br />
npm run dev
